First, update and refresh `howebrew`.
```
brew update
```

# Install `mpd`, the music player daemon
```
brew install mpd
```

# Install `mpc`, a command line music player client
```
brew install mpc
```

# Instal `ncmpcpp`, a powerful yet lightweight `mpd` client
```
brew install ncmpcpp
```
