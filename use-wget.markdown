Download an entire website with all its content and media using `wget`.

# Description
```
    wget --recursive \
        --no-clobber \
        --page-requisites \
        --html-extension \
        --convert-links \
        --domains website.tld \
        --no-parent \
        	http://website.tld/foo/bar
```

# Simple
```
wget -r -nc -p -E -k -D website.tld -np http://website.tld/foo/bar
```
